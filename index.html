Welcome to vi·si·cal·i·ty (visual + musicality)


vi·si·cal·i·ty is a web-based music visualizer utilizing Web Audio API and D3.js.


Thanks for visiting 🙂

Keyboard Controls

In addtion to on-screen controls, vi·si·cal·i·ty supports a wide range of keyboard shortcuts:

Key Action
Spacebar Play / Pause Audio
Left-Arrow Seek audio 5 seconds backwards
Right-Arrow Seek audio 5 seconds forward
Up-Arrow Increase gain (volume) by 10 percent
Down-arrow Decrease gain (volume) by 10 percent
F Enter / Exit fullscreen mode
W Previous visualizer design, based on associated number
S Next visualizer design, based on associated number
A Previous visualizer color, based on associated number
D Next visualizer color, based on associated number
R Set random three-color background gradient
Q Rotate background left 45 degrees
E Rotate background right 45 degrees
